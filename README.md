# Saarthi AI 2.0

## Inclusive AI-Powered Government Scheme Recommendation Platform

Saarthi AI 2.0 helps citizens—especially visually impaired, hearing-impaired, and low-literacy users—discover government schemes they may be eligible for through an accessible AI-powered interface.

## Problem
Many citizens struggle to identify which government schemes they qualify for due to complex eligibility criteria, language barriers, and accessibility challenges.

## Solution
Saarthi AI 2.0 provides personalized scheme recommendations using an accessibility-first AI workflow with voice assistance, vibration cues, and simplified explanations.

## Implemented Features
- AI-based scheme recommendation
- Voice read-aloud assistance (Text-to-Speech)
- High-contrast accessible interface
- Vibration feedback for supported devices
- Responsive React frontend

## Accessibility Features
- Screen-reader friendly structure
- Voice output for visually impaired users
- Vibration cues for hearing-impaired users
- Easy-read interface design

## Tech Stack
- React + Vite
- Node.js
- FastAPI
- Supabase
- Web Speech API

## Architecture
React → Node.js → FastAPI → Supabase

## Run Locally

bash
cd frontend
npm install
npm run dev


Open: http://localhost:5173
